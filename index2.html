Question-1: What are the different ways to select an element in the DOM?

Ans:There are multiple ways to select elements in the DOM.Those are given below:-
1. Selecting by ID (getElementById).
2. Selecting by Class Name (getElementsByClassName).
3. Selecting by Tag Name (getElementsByTagName).
4. Selecting with CSS Selectors (querySelector).
5. Selecting Multiple Elements with CSS Selectors (querySelectorAll).
6. Selecting by name Attribute (getElementsByName).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:Differences Between innerHTML, innerText, and textContent in JavaScript.
Here's how they are different: 1.Use innerHTML When you need to add or modify HTML structure.
2.Use innerText When you only need the visible text .
3.Use textContent When you need all text including hidden content and for better performance.

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a technique where we can attach a single event listener to a parent element instead of adding event listeners to multiple child elements. Due to better performance ,handling dynamic elements and less code.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a process where an event starts from the element that triggered it and then travels up (bubbling) through its parent elements until it reaches the end of the html root.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: We can create, add, and remove elements dynamically using the DOM (Document Object Model).
Those are:- 1.we can create an element using the document.createElement() method.
2.There are several ways to Add Elements .Those given below:-
a) Append an Element to the End (appendChild).
b) Insert an Element Before Another (insertBefore).
c) Use append() allows adding multiple elements and even text.
d) Use prepend() Adds the new element as the first child.
3. We can remove an element using the following methods:-
a. Using remove() to Remove the element from the DOM.
b. Using removeChild() to Remove a specific child from its parent.
